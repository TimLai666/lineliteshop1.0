# Mobile Cinematic Refresh Decisions

Date: 2026-04-15

## Start Questionnaire

- Output medium: Website / Web app
- Start mode: Screenshot
- Image placeholders: No
- Niche and page list: LINE LIFF mobile food ordering app with `UserRegister`, `OrderFood`, and `MyOrders`

## Chosen Film Language

- Director: Wong Kar-wai
- Film: `In the Mood for Love`
- Why this fit:
  - The existing orange-led palette already supports warm tungsten light without a theme-color reset.
  - The app is mobile-first, so close framing, intimate spacing, and corridor-like vertical rhythm translate well.
  - The user asked for beauty, not a colder enterprise redesign. This film language raises tactility and mood while staying usable.

## Research Notes

- `In the Mood for Love` is defined by tight framing, warm practical light, soft diffusion, and frames-within-frames.
- The visual tension comes from restraint, not spectacle: still compositions, selective glow, compressed space, and emotionally charged surfaces.
- Premium coffeehouse and ordering references reinforce warmer materials, layered textures, and clearer pickup / action zones.

## Previous-work Audit

- No prior cinematic site output for this repo was reviewed.

## Shell-ban List

- No flat white feed-like mobile shell
- No generic SaaS gradient hero followed by ordinary cards
- No heavy neon or sci-fi glass treatment
- No repeated pill + card pattern without framing depth

## Primary Composition Family

- Panoramic slab on cards plus corridor-style vertical flow for pages

## Reference Protocol

- Reference A contributes: the user screenshot contributes clean card framing, generous product imagery, and a centered focal hierarchy
- What will not be copied: flat social-card styling, generic profile-card rhythm, plain white lower panels with no atmosphere

## Site Cinematic Grammar

- Shell logic: mobile-first vertical sequence with framed surfaces floating over a warm atmospheric backdrop
- Navigation posture: sticky top bars and tab rails behave like a quiet title card, not a utility toolbar
- Framing discipline: every major surface gets a clear edge treatment, inner stroke, and shadow depth
- Density cadence: image / summary / action / modal should alternate between breath and compression
- Recurring materials: porcelain-white surfaces, amber glow, soft blur, inset borders, light grain-like gradients
- Motion posture: restrained; mostly hover / active lift and modal depth, not constant choreography
